Computes the operator-multivector application.

Loosely, performs . However, the details of operation vary according to the values of alpha and beta. Specifically - if beta == 0, apply() must overwrite Y, so that any values in Y (including NaNs) are ignored. - if alpha == 0, apply() may short-circuit the operator, so that any values in X (including NaNs) are ignored.
